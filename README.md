### About
Omnifood is a responsive web application designed for food lovers. It provides a seamless experience for browsing menus, ordering meals, and tracking deliveries. The project emphasizes modern UI/UX design, responsive layouts, and accessibility.

---

### Features
- Browse meals by category  
- Responsive design for mobile and desktop  
- Smooth animations and interactive UI  
- Contact form with validation  
- Easy-to-use navigation  

---

### Usage

- Open the website in your browser  
- Explore the menu, select meals, and interact with the site  
- Use the contact form to submit inquiries  
- Fully responsive for tablets and mobile devices
